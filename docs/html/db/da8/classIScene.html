Detailed Description

Abstract base interface for all scene implementations.

Defines the interface that all scenes must implement to work with the SceneManager. A scene represents a distinct state or screen in the game, such as menu, lobby, gameplay, or end screen.

Note
All derived classes must implement the pure virtual methods.
