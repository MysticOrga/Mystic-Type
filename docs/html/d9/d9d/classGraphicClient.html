Detailed Description

Main graphics client application.

Manages the entire graphical client application including:

  • Window initialization and management
  • Scene management and transitions
  • Network communication
  • Game loop execution

The GraphicClient is responsible for coordinating between the Raylib window, the network client, and the scene manager to provide a complete game experience.
